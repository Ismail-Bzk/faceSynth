```bash
npm run dev

# 3. Open

# 4. Build

```

## 📝 Customisation minimalistes

**Pour tester rapidement (5 min):**

1. Couleurs : `tailwind.config.ts` → modifiez `colors`
2. Textes : `app/constants/site.ts` → `SITE_CONFIG` + `HERO_COPY`
3. Favicon : remplacez `public/favicon.ico`

**C'est tout pour une démo !**
